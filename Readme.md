# wrap_libusb
`wrap_libusb` is an Eiffel binding of [libusb C API](https://libusb.info/) 
using [WrapC](https://github.com/eiffel-wrap-c/WrapC) tool.

## Requirements 


*  [WrapC](https://github.com/eiffel-wrap-c/WrapC) tool.
*  [libusb C API](https://libusb.info/).
